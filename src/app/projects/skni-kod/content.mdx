This project is a complete rewrite of
[Student Research Group of Computer Science — ‘Code’](https://skni.prz.edu.pl/)
website, featuring [OAuth 2.0](https://oauth.net/2/) authentication and a blog
with content management capabilities.

## My role

The old SKNI Kod website built with [Vue.js 2](https://v2.vuejs.org/) and
[Django](https://djangoproject.com/) frameworks was created by the more
experienced members of the group. As they graduated, the project gradually began
to receive fewer updates until it was eventually no longer maintained.

Since I had to create a web application for my BSc thesis — ‘Use of
[Next.js](https://nextjs.org/) Framework for Server-Side Application
Development’ — I decided to seize an opportunity and build a new website for
SKNI Kod using modern technologies.

Official brand assets and list of requirements the website should meet were
provided by group's management. I implemented the most crucial features such as
authentication and content management first, while working on my thesis.
Features that were out of scope of the thesis are currently a work in progress.
