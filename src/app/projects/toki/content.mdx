Toki is a minimalist count up timer with few conveniences, such as progress
tracking with milestones, as well as counting any other values over time.

<Image alt="Toki's homepage view." src={tokiHomePage} />

## Features

- Any number of counters
- Milestones
- Figures
- Refresh-proof, persistent settings
- Export configuration to a file
- Import configuration from a file
- Multiple themes (dark, light, pink and system)

<Image alt="Toki's list view." src={tokiList} />

## My role

One of my relatives reached out to me after failing to find any count up timer
applications that would meet their requirements. I built Toki myself from start
to finish, having only guidelines for the features that the application should
have.
