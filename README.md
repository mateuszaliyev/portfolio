# Portfolio

## Getting Started

### Prerequisites

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/)
- [pnpm](https://pnpm.io/)

### Installation

Clone the repository.

```bash
git clone https://github.com/mateuszaliyev/portfolio.git
# or
git clone git@github.com:mateuszaliyev/portfolio.git
```

Go to the project directory.

```bash
cd portfolio
```

Install dependencies.

```bash
pnpm install
```

## Usage

Run the development server.

```bash
pnpm dev
```

Lint code with [ESLint](https://eslint.org/).

```bash
pnpm lint
```

Fix any existing errors.

```bash
pnpm lint --fix
```

Format code with [Prettier](https://prettier.io/).

```bash
pnpm format
```

Build application for production.

```bash
pnpm build
```

Start production server.

```bash
pnpm start
```
